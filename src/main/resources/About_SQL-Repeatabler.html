SQL-Repeatabler ${project.version}

This tool enhances SQL-Scripts (MS flavour), which are generated by lo-code
platform 'Appian', to make them runable repeatedly.
I.e. a 'create table' should not generate a 'table already exists'
error when running another time.
The converted scripts are intended to be used in a deployment
system that wants to execute them more than once.
This tool only covers the SQL commands that Appian has
confronted us with so far and will only be enhanced as needed.
